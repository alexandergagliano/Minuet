# Minuet Browser 

A high-performance, WebGL-powered visualization of galaxy embeddings.

## Usage

### Starting the Server
```bash
cd Minuet/browser
python app.py
```

The application will be available at `http://localhost:5002`

### Navigation
- **Left click + drag**: Pan the view
- **Right click + drag**: Rotate the view  
- **Scroll wheel**: Zoom in/out
- **Click on galaxy**: View detailed predictions
